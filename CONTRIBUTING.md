# Contributing to Agent Messenger

This is an open source project in early development. Contributions welcome.

## How to Contribute

1. Fork the repository
2. Create a branch (`feature/`, `fix/`, etc.)
3. Write code + tests
4. Submit a PR with a clear description

## Code Style

### Go (Server)
- `gofmt` and `goimports`
- Follow [Effective Go](https://golang.org/doc/effective_go)
- Comment public functions

### TypeScript (OpenClaw Plugin + WebChat)
- Follow existing patterns in the codebase
- Write tests for new functionality

### Swift (iOS) / Kotlin (Android)
- Follow platform conventions
- SwiftUI / Jetpack Compose respectively

## Commit Messages

```
type(scope): brief description
```

Types: `feat`, `fix`, `docs`, `refactor`, `test`, `chore`

## Security Vulnerabilities

Open a GitHub issue. This is open source — public disclosure helps everyone.

## Code of Conduct

Be respectful. Be constructive.
